Span

For an element that offers no semantic information about the content inside and provides no styling change, or any opther visual change to speak of,the span element is one of the most useful elements in our HTML toolbox. When I wrap text with an opening <span> and closing </span>,I am simply providing a hook-one that allows me to add styles (by adding a class attribute and using CSS to define the look of that class).

Be careful to use the span tag to replace elements such as strong, em cite and blockquote.These elements are useful as they have a meaning and describe the structure.
