Stylesheets

What is Stylesheets

CSS stands for cascading Style Sheets;This language allows to change formatting of elements such as text,size,color,bolding, background colors,position on the page and much more.

Anatomy of the rule

p
{
font-family: Arial,helveatica, sans-seriff;
color: blue; font-size=25;
}

The whole thing is called a rule.p is the selector.Curly braces surround p are declarations.To apply the CSS we need to link the code in the HTml file.

Linking CSS to a page

 

<link href="style.css" rel="stylesheet"type="text/css" />

Embedded Style

Embedded style sheets allows you to add at start of
webpage a section that sets out all styling to be used in the page

All paragraphs on this page displayed in blue Aerial

Inline Style

The style attribute is added inside h1 tag to give heading color it contains one or more declarations A declaration is made up of two parts.

  • a property and
  • its value

In our heading example <h1 style="color:red;">inline Style</h1> the declaration is color:red(color is the property and red is the value). A span tag allows you to highlight a section to which you want to apply a style to.
