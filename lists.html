Dictionary List

Ordered list
Used to group a collection of items together in a list.
Each list item is defined by a li element,
which suggests an order of importance or sequence,
as ol is an abbreviation of ordered list. The ol may be
used to mark up a series of steps that someone has to undertake, a table of contents, or a numbered list of references.
UnOrdered List
The ul element, the name for which is an abbreviation of unordered list,
is used to group a collection of items together in a list, but in a way that
doesn't suggest an order of precedence or importance. The ul might be used for
anything from a simple shopping list to a collection of links.
Dictionary list
Dictionary term
If you want to list a series of items that essentially have a title and a description of some kind (that is, each item has two parts), use the definition list dl element. This element contains a series of definition term (dt) and definition description (dd) pairings
