Box Model

A block-level element can be manipulated using a combination of margin, border, padding, height, and width values. This is known as the box model.

Setting width and height

The width can be set by using the width property:

width: 200px;


The height can be set by using the height property:


height: 300px;

Borders


You can use block-level elements to apply a range of border effects, including:
  • different border thicknesses
  • various border styles (solid lines, dotted lines, and more)
  • a range of border colors


The CSS properties we can use to achieve these effects are:
  • border-width
  • border-style
  • border-color

Padding

Padding adds some space between the border and the text.

Margin

You can modify how much space surrounds a block by modifying the setting of the margin.
